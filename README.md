Finance Tracker:

Finance Tracker is a simple Financial Tracker tool that's build with ASP.NET Core Razor Pages and SQLite. 
It allows you to track income, expenses, monthly spending, transactions history and user selected monthly summary.
This project was mainly build to learn core web page development and backend concepts using C# and .NET


Features:
- Add Income and Expenses 
- Categorised transactions
- View all Transactions by date
- Monthly Summary showing (Total Income, Total Expenses, Monthly Net)
- Validation for input values
- SQL datebase storage
- Simple and clean UI. 


Tech Stack:

- C#
- .NET 8 / ASP.NET Core Razor Pages
- Entity Framework Core
- SQLite
- HTML / Bootstrap (Default Style)


To Run:

1. .NET SDK Installed
2. Download repo (git clone <repo-url>)
3. cd FinanceTracker
4. dotnet run
5. Copy and paste local host URL in your web browser


Learning Objectives:

- How Razor pages works with using folders and @pages.
- Working on both backend and frontend development of a web page.
- Debugging run time errors and stack traces.
- SQLite basics.
